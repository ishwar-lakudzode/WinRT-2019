Using this application we an draw line, rectangle, circle(Ellips) etc.
1. Presh Shift+L to start line drawing, then press mouse LButton down and drag mouse in window same like we do in MS Paint
2. Presh Shift+R to start rectangle drawing, then press mouse LButton down and drag mouse in window same like we do in MS Paint
3. Presh Shift+E to start circle(Ellips) drawing, then press mouse LButton down and drag mouse in window same like we do in MS Paint
4. Presh Shift+E to start Animation
   In this press + button to slow down animation same like volume
   press - button to speed up animation.

you need to press multiple time + and - button for speed control
